    • How To Disable Program Compatibility Assistant (PCA) In Windows 7 & Vista

    1. Type GPEDIT.MSC in start menu search box or run and press enter.

    2. Navigate to:
      User Configuration -> Administrative Templates -> Windows Components -> Application Compatibility

    3. Double-click on Turn off Program Compatibility Assistant.

    4. Select Enabled.

    5. Click OK.

    Now the Program Compatibility Assistant (PCA) dialog box will never pop-up again.
