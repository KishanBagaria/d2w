    Here is a curated set of start buttons that can be used with CustomizerGod, Windows 7 Start Orb Changer or Windows 8.1 Start Button Changer.

    If you don't know how to extract 7z or rar files, use 7-Zip. If the start button is for Windows 7 (there are three buttons in one image) and you want to use it in Windows 8.1+, use Windows 7 to Windows 8.1 Start Button Converter.

    If the downloaded contains only the exe file without the original bitmap, you'll have to extract the bitmap:

    1. Download and run CustomizerGod.
    2. Press Ctrl+O and open the exe file that you downloaded earlier.
    3. Select the new Bitmaps option that gets listed in the left-hand pane.
    4. Select the start button and click Export Selected Resources.
