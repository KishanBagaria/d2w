    • How To Create Virtual Drives Of Folders

    Visual Subst is a free tiny portable tool for Windows that allows you to create virtual drives of folders with ease. It is the GUI version of the subst command in Windows.

    How to use:

    1. Run the tool.

    2. Click on the icon near to the text field:

    3. Select the folder that you want to make a virtual drive and click OK.

    4. Select your desired drive letter & click on the Add icon:

    To delete any virtual drive, just select it & click on the Cross icon.
